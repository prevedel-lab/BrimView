Your browser is not supported

Please first make sure you have the latest version of the browser. For the major browsers:

  • We currently support only the latest version of Chrome and Edge out of the box.
  • Firefox can be used by first enabling JS Promise integration. How to do it?
    Enabling JSPI in Firefox
    • Make sure you have the latest version of Firefox installed
    • Open a new tab and navigate to about:config
    • Search for wasm_js_promise_integration and set it to true
    • Restart Firefox
  • Safari can't work as it doesn't support JS Promise integration.

Alternatively, you can download the app for Windows or Mac.

If you believe your are getting this message despite your browser supporting JSPI, you can open a GitHub issue
